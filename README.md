# Location Tracking Api

Backend system that stores records which contain device location and other important information, depending on the device’s protocol. It works as a complete GPS tracking platform, managing real-time positions, device data, and historical records for monitoring and tracking purposes.

# Architecture layers
- WEB (Controllers)
- Application (Services)
- Infrastructure (DbContext and Repositories)
- Domain (Entities)
- Tests (Web, Application, Infrastructure, Domain)

# Technologies and Tools
- .NET 9.0 
- ASP.NET Core Web API
- Entity Framework Core 
- PostgreSQL Database 
- Scalar for API testing
- XUnit testing
